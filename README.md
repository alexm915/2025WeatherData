            su - alex -c "/project/idc/cpp/start.sh
            #意思是切换到普通用户alex去执行此脚本

## 2- 程序的监控与调度
要解决的问题：
    1. 让监控程序周期性运行一些调度任务；
    2. 检查服务程序是否活着；
    3. 如果服务程序已经终止，重启它。
两种服务程序:
    1. 周期性运行的：如，每一分钟要运行一次，生成一次测试数据； ===> 周期性启动
    2. 常驻服务：如，网络通讯服务，不间断7x24运行 ===> 正常/异常终止后, 重启它。
